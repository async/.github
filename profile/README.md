# Async

Async builds small, inspectable tools for moving from prototype code to production-shaped software without throwing the prototype away.

## Projects

- [db](https://github.com/async/db) - Gradual data workflow from JSON fixtures to typed APIs, local runtimes, writable stores, and durable persistence.
- [web](https://github.com/async/web) - Web app, router, and Request -> Response runtime packages for browser, API, edge, and data-backed apps.
- [pipeline](https://github.com/async/pipeline) - Local-first TypeScript pipelines for task graphs, run evidence, cache inputs, and CI workflows.
- [framework](https://github.com/async/framework) - Signal-based framework experiments for reactive web apps, async handlers, and runtime-first UI.
- [framework-demo](https://github.com/async/framework-demo) - Public demos and runtime examples for the framework work.

The older `async-framework` organization is kept only for redirects and historical references.
